# Online Shopping Cart System (Python OOP)

## Overview
A console-based shopping cart system built using Python and Object-Oriented Programming.

## Features
- Add Product
- Remove Product
- Update Quantity
- Display Cart
- Calculate Total
- Empty Cart
- Stock Validation

## OOP Concepts Used
- Classes & Objects
- Composition
- Dictionaries
- Methods

## Project Structure
- Product
- User
- ShoppingCart

## How to Run
python shopping_cart.py
